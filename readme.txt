Core functionality plugin for the Greenova WordPress theme.

== Description ==

Greenova Core is a companion plugin designed specifically for the Greenova WordPress theme. It provides essential features and functionality including custom post types, custom widgets, and page builder integrations.

**Features:**

* Custom Post Types (Services, Projects, Team Members, etc.)
* Custom Widgets for sidebar and footer areas
* Elementor widgets and extensions
* WPBakery Page Builder (Visual Composer) modules
* Demo content importer
* Custom meta boxes for enhanced content management
* Flaticon integration for Visual Composer
* Social share functionality

**Page Builder Support:**

* Full Elementor integration with custom widgets
* WPBakery Page Builder (Visual Composer) modules
* Compatible with popular page builders

**Custom Post Types:**

* Services - Showcase your business services
* Projects/Portfolio - Display your work
* Team Members - Introduce your team
* Testimonials - Customer reviews and feedback

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/greenova-core` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. The plugin works seamlessly with the Greenova theme and requires the theme to be active.
4. Configure the plugin settings from the WordPress admin panel.

== Frequently Asked Questions ==

= Does this plugin work with any theme? =

No, this plugin is specifically designed to work with the Greenova WordPress theme. It may not function properly with other themes.

= Is this plugin required for the Greenova theme? =

While the Greenova theme can function without this plugin, it is highly recommended to install it to unlock all theme features and functionality.

= Does it support Elementor? =

Yes, the plugin includes custom Elementor widgets specifically designed for the Greenova theme.

= Does it support WPBakery Page Builder? =

Yes, the plugin includes custom Visual Composer modules for WPBakery Page Builder.
